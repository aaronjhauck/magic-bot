#### Install

* `$ git clone https://github.com/aaronjhauck/magic-bot.git`
* `$ npm install`

#### Configure

* install docker
* install docker-compose

#### Test

* `$ npm test`
* `$ ./tests/tests.js`

#### Run
* `$ docker-compose build`
* `$ docker-compose up -d`


_magic-bot uses nodemon -- any hot-fixes made locally will be immediately deployed to the container_
